## Configuration
You will need to add a file `config/keys.js` with the following information:
```javascript
module.exports = {
  mongoURI: "YOUR_MONGO_URI_HERE",
  secretOrKey: "secret"
};
```

## Getting Set up: install dependencies
```bash
npm install
npm run client-install
```

## Starting the server


```bash
npm run dev
```
